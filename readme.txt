== Description ==

Adds a customizable search widget allowing you to change the widget title,
search field length, and button value.

== Installation ==


1. Upload the `better-search-widget` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the widget to a dynamic sidebar.

== Frequently Asked Questions ==

= Why? =

The built-in Search widget sucks. It doesn't follow the format of the other
widgets and allows no customization.
